## 💡 기획 의도 : 왜 '그냥퀵위젯'인가요?

사용자 작업의 편의를 위해 매크로 등을 특정 키에 할당하여 사용하는 훌륭한 제품들이 시중에 많이 있습니다. 물론 더 세련되고 전문적인 기능을 제공하는 프로그램들도 존재합니다.

하지만 그와 비교했을 때, 이 프로그램은 **"전문적인 지식이 없는 일반 사용자들도 그와 비슷한 수준의 쾌적한 데스크탑 경험과 업무 편의성을 누릴 수 있었으면 좋겠다"**는 생각에서 출발했습니다. 복잡한 설정 없이 직관적으로 사용할 수 있도록, 데스크탑 작업 표시줄을 확장한 개념의 '위젯 형식'으로 구현한 프로그램을 소개합니다.

**"있는 듯 없는 듯, 하지만 필요할 땐 확실하게"**
보통 개발자 입장에서는 자신이 만든 프로그램이 화면에서 화려하게 눈에 띄고 주목받기를 원합니다. 하지만 이 위젯은 철저히 '보조 프로그램'으로서, 사용자의 본래 작업 환경을 절대 방해하지 않고 화면 구석에 조용히 머물기를 바라는 마음으로 설계했습니다. 

다만, 너무 조용히 숨어있다 보니 사용자가 위젯을 띄워두고도 어디 있는지 찾지 못해 앱을 다시 실행하게 되는 역설적인 상황이 종종 발생했습니다. 이를 해결하기 위해 **'투명도 조절'과 '배경 이펙트'** 기능을 추가했습니다. 평소에는 바탕화면에 스며들듯 투명하게 숨어있다가도, 시각적 효과를 통해 필요할 때는 사용자가 단번에 위치를 찾을 수 있도록 세심한 배려를 담았습니다.

<br>

## 📖 100% 바이브 코딩(Vibe Coding) 그냥개발자K

이 프로그램은 단 한 줄의 코드도 직접 타이핑하지 않고 **100% '바이브 코딩'**으로 만들어졌습니다. 
비개발자의 시선으로 "이런 위젯이 있으면 좋겠다"는 아이디어를 제미나이(Gemini)와 끊임없이 대화하고 복붙(?)하는 과정만으로 완성했습니다. (Claude Code나 Antigravity 같은 개발용 툴의 전문적인 기능은 전혀 사용하지 않은 순수 대화의 결과물입니다!)

기획부터 앱 배포까지, AI와 함께 맨땅에 헤딩하며 만들어낸 이 즐거운 도전의 결과물이 여러분의 데스크탑 환경에 작은 편안함을 주길 바랍니다.

## ✨ 핵심 기능 안내 (Features)

### 🖱️ 1. 위젯 바(Widget Bar) 직접 제어 및 직관적인 조작
위젯 바 위에서 마우스 조작만으로 거의 모든 제어가 가능합니다.

* **크기 조절:** 위젯 바 위에 마우스를 올리고 `Ctrl + 마우스 휠`을 굴려 즉각적으로 크기를 변경합니다.
* **페이징(페이지 이동):** 위젯 바 위에서 `마우스 휠`만 굴리면 다음 페이지로 부드럽게 넘어갑니다.
* **버튼 제어:** * 드래그 앤 드롭: 바탕화면의 바로가기, `.exe` 파일, 폴더 등을 끌어다 놓으면 즉시 버튼으로 등록됩니다.
  * 좌클릭: 등록된 내용 실행 (빈 버튼 클릭 시 새 버튼 편집창 실행)
  * 우클릭: 등록된 버튼 내용 수정 및 삭제

### ⚙️ 2. 위젯 설정 > 버튼 배치 관리
내 입맛에 맞게 총 3페이지, 30개의 버튼 공간을 100% 활용하세요. 버튼 순서 변경, 페이지 간 이동은 물론 삭제하기 아까운 버튼을 담아두는 '보관함' 기능도 제공합니다.

### 🛠️ 3. 버튼 편집 3가지 강력한 유형

* **유형 A: 파일 / 폴더 / 프로그램 실행**
  * 빈 공간에 드래그 앤 드롭하면 이름/경로/아이콘이 자동 추출됩니다. 커스텀 이미지 파일을 드롭하여 나만의 아이콘으로 덮어쓸 수도 있습니다.
* **유형 B: 웹사이트 열기**
  * URL 주소를 입력하면 자동으로 웹사이트 유형으로 변경되며, 원하는 브라우저와 실행 방식(새 창/탭)을 지정할 수 있습니다.
* **유형 C: 워크플로우 (자동화 루틴) 🔥**
  * **버튼 클릭 한 번으로 여러 동작을 순차적으로 실행하는 매크로 기능입니다.**

> **[워크플로우 상세 블록 설명]**
> * **🎯 실행 타겟 창 지정:** 현재 띄워진 창 목록을 불러와 특정 창을 타겟으로 고정할 수 있습니다. 타겟이 지정되면 오직 해당 프로그램 창이 활성화되어 있을 때만 동작하며, 엉뚱한 곳에서의 오작동을 막아줍니다. (단, 프로그램이 꺼져있을 때 대신 실행시켜 주지는 않습니다.) 타겟을 비워두면 현재 활성화된 아무 창에서나 범용적으로 동작합니다.
> * **📝 텍스트 입력:** 여기에 입력한 텍스트를 즉시 '클립보드(복사 상태)'로 만듭니다. 이후 '키 입력(Ctrl+V)' 블록과 조합해 아이디나 비밀번호 등을 자동으로 붙여넣을 수 있습니다.
> * **⌨️ 키 입력:** 윈도우 우선순위 시스템 단축키(Win 조합)를 제외한 실제 키보드 단축키를 지정합니다. (예: 붙여넣기, 프로그램 내장 단축키 등)
> * **⏳ 대기:** 동작 블록들 사이에 시간을 벌어주어 다음 동작이 씹히지 않고 안정적으로 실행되도록 돕습니다. (예: 1000 = 1초)
> * **💻 시스템 제어:** 일반 키 입력으로는 막혀있는 강력한 윈도우 시스템 단축키(바탕화면 보기 등)를 목록에서 쉽게 선택해 사용할 수 있습니다.
> * **🚀 프로그램 실행:** 현재 실행 중인 프로그램 목록에서 고르거나, 직접 경로를 찾아 지정하여 프로그램이나 URL을 실행시킵니다.
> * **🎵 미디어 제어:** 음악 재생, 일시정지, 볼륨 조절 등 미디어 동작을 쉽게 제어합니다.

#### 💡 워크플로우(자동화) 구성 예시

* **예시 1) 🚨 상사 등장! 0.1초 긴급 회피 (Boss Key) 모드**
  월급루팡(?) 중 누군가 다가오는 인기척이 들릴 때 버튼 하나로 대처하세요!
  1. `[미디어 제어]` : 볼륨 음소거 (유튜브 소리부터 긴급 차단!)
  2. `[시스템 제어]` : 바탕화면 보기 (모든 딴짓 창을 0.1초 만에 최소화)
  3. `[대기]` : 300 (0.3초 딜레이)
  4. `[프로그램 실행]` : 엑셀(Excel) 프로그램이나 업무 폴더 경로 지정 (열심히 일하는 척 위장 완료!)

* **예시 2) 🌍 드래그한 외국어 1초 만에 번역기 돌리기**
  업무 중이나 웹서핑 중 모르는 외국어가 나왔을 때, 텍스트를 드래그만 해두고 이 버튼을 누르세요.
  1. `[키 입력]` : Ctrl + C (드래그해 둔 텍스트 복사)
  2. `[프로그램 실행]` : https://papago.naver.com (파파고 번역 웹사이트 열기)
  3. `[대기]` : 1000 (번역 페이지가 로딩될 때까지 1초 대기)
  4. `[키 입력]` : Ctrl + V (번역기 입력창에 복사한 텍스트 자동 붙여넣기)

### 🎛️ 4. 위젯 기본 설정 및 데이터 관리
* 설정 창의 **[단축키 활성화]** 옵션을 켜면 위젯의 버튼들을 마우스 클릭 없이 전역 단축키로 실행할 수 있습니다. (단축키 충돌에 유의하세요!)
* 힘들게 만든 나만의 위젯 환경과 워크플로우 버튼을 파일로 묶어 완벽하게 **전체 백업/복원** 하거나, 동료에게 **공유(Export/Import)** 할 수 있습니다.

## 🚨 설치 및 실행 가이드 (필독: 보안 경고창 관련)

개인 개발자가 무료로 배포하는 인디 프로그램의 특성상 설치 시 윈도우 보안 경고가 나타날 수 있습니다. **이는 서명되지 않은 실행 파일(.exe)에 대한 윈도우의 기본 방어 기제일 뿐, 바이러스가 아닙니다.**

* **파란색 창(SmartScreen)이 뜰 때:** 텍스트 중간의 **[추가 정보]** 클릭 👉 새로 생긴 우측 하단의 **[실행]** 버튼 클릭
* **백신에서 차단할 때:** 내부 매크로(PowerShell) 제어 로직을 오진할 수 있습니다. 백신 설정에서 `GN-QuickWidget-Setup.exe` 파일이나 설치 폴더를 **[검사 예외 / 신뢰하는 프로그램]**으로 등록해 주세요.
