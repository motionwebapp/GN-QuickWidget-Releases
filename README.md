# 🚀 GN-QuickWidget (그냥-퀵위젯)

**"데스크톱 환경의 한계를 넘는 나만의 초소형 생산성 컨트롤 타워"**

GN-QuickWidget은 바탕화면의 공간을 차지하지 않으면서도, 사용자가 가장 자주 쓰는 프로그램, 웹사이트, 그리고 복잡한 반복 작업(매크로)을 단 한 번의 클릭과 단축키로 실행할 수 있게 해주는 스마트 플로팅 위젯입니다.

## 🎨 1. 압도적인 UI/UX (데스크톱에 스며드는 디자인)

* **🧲 스마트 자석 스냅:** 위젯을 모니터 가장자리로 가져가면 화면 경계에 빈틈없이 쫀득하게 밀착됩니다. 공간이 필요할 땐 자유 모드로 화면 어디에나 띄워둘 수 있습니다.
* **💊 미니(축소) 모드:** 작업에 집중해야 할 때 버튼을 클릭하면, 위젯이 작은 알약 형태의 컨트롤 바로 축소되어 화면 방해를 최소화합니다.
* **✨ 감성적인 비주얼 & 다크모드:** 은은하게 회전하는 네온 Glow 이펙트와 투명도 조절 기능을 지원하며, 눈이 편안한 '다크모드'를 통해 어떤 배경화면에서도 고급스럽게 어우러집니다.
* **📱 스와이프 & 휠 페이지 이동:** 총 3개의 페이지(최대 30개 버튼)를 지원하며, 마우스 휠이나 화면 스와이프로 부드럽게 페이지를 넘나들 수 있습니다.

## ⚡ 2. 핵심 기능 3대장 (무엇이든 연결합니다)

단순한 바로가기를 넘어, 3가지의 강력한 기능 유형을 하나의 폼에서 직관적으로 설정할 수 있습니다.

1. **📁 파일 / 폴더 / 프로그램 열기**
* **스마트 드래그 앤 드롭:** 바탕화면의 아이콘(`.lnk`)을 끌어다 놓기만 하면, 윈도우가 인식하는 가장 선명한 원본 아이콘과 실제 `.exe` 실행 경로를 완벽하게 역추적하여 자동 등록합니다.


2. **🌐 웹사이트 (URL) 다이렉트 실행**
* **도메인 자동 인식:** `naver.com`처럼 도메인만 입력해도 자동으로 URL로 인식하여 해당 사이트의 파비콘(로고)을 즉시 가져옵니다.
* **브라우저 지정:** Chrome, Edge, Whale 등 특정 사이트를 어떤 브라우저로 열지, 새 창으로 열지 세밀하게 지정할 수 있습니다.


3. **🤖 그냥 워크플로우 (강력한 자동화 매크로)**
* **타겟 창 포커싱:** 켜져 있는 특정 프로그램 창을 화면 맨 앞으로 강제로 끌어온 뒤 동작을 실행합니다.
* **블록 조립형 매크로:** '텍스트 자동 타이핑', '단축키 입력', '미디어 제어(볼륨/재생)', '시스템 제어(화면 잠금, 바탕화면 보기 등)'를 블록처럼 조립해 나만의 자동화 루틴을 만듭니다.



## ⚙️ 3. 직관적이고 강력한 관리 환경 (설정창)

* **네이티브 수준의 모던 설정창:** 16:10 와이드 비율의 넓고 쾌적한 화면, 카드 형태로 논리적으로 분리된 깔끔한 UI를 제공합니다.
* **버튼 배치 관리 & 보관함 (Archive):** 스마트폰 홈 화면을 꾸미듯 드래그 앤 드롭으로 버튼 위치를 자유롭게 바꾸고, 당장 안 쓰는 버튼은 '보관함'에 넣어둘 수 있습니다.
* **버튼 단건 공유 (클립보드):** 정성껏 세팅한 매크로나 버튼 설정을 텍스트 형태로 복사하여 다른 기기나 동료에게 쉽게 공유하고 붙여넣을 수 있습니다.
* **전체 백업 & 복원:** 모든 세팅과 아이콘 정보를 JSON 파일 하나로 완벽하게 백업하고 복원합니다.

## 💡 4. 100% 활용 시나리오 (이런 분들께 추천합니다)

* **💻 듀얼 모니터를 쓰는 헤비 유저:**
* 자석 스냅으로 모니터 구석에 위젯을 짱박아(?) 두고, 마우스 이동 없이 전역 단축키를 눌러 자주 쓰는 툴을 즉시 호출합니다.


* **🏢 반복 업무가 많은 직장인 (워크플로우 활용):**
* **빠른 인사말 매크로:** 메신저 창을 타겟팅하여 "안녕하세요, 확인 후 회신드리겠습니다." 텍스트와 `Enter` 키를 0.1초 만에 자동 입력.
* **퇴근 전 원클릭 셧다운:** [바탕화면 보기] -> [PC 절전 모드] 또는 [시스템 종료] 매크로.


* **🛠️ 효율을 중시하는 개발자 & 디자이너:**
* 로컬 서버 폴더, 특정 프로젝트의 GitHub 주소(Chrome 강제 실행), 피그마 앱 등을 한 페이지에 모아두고 프로젝트 컨트롤 패널로 사용합니다.

## 🚨 설치 및 실행 가이드 (필독: 보안 경고창 관련)

개인 개발자가 무료로 배포하는 인디 프로그램의 특성상 설치 시 윈도우 보안 경고가 나타날 수 있습니다. **이는 서명되지 않은 실행 파일(.exe)에 대한 윈도우의 기본 방어 기제일 뿐, 바이러스가 아닙니다.**

* **파란색 창(SmartScreen)이 뜰 때:** 텍스트 중간의 **[추가 정보]** 클릭 👉 새로 생긴 우측 하단의 **[실행]** 버튼 클릭
* **백신에서 차단할 때:** 내부 매크로(PowerShell) 제어 로직을 오진할 수 있습니다. 백신 설정에서 `GN-QuickWidget-Setup.exe` 파일이나 설치 폴더를 **[검사 예외 / 신뢰하는 프로그램]**으로 등록해 주세요.
